Lawn Tennis Game Rules

Set Rules

  • Each set is played for a maximum of 3 games a side, with a standard tie-break if the score reaches 3-3.

Timing Rules

  • 15 seconds gap between the first and second serve.
  • 25 seconds gap between two serves.
  • 90 seconds gap between changeovers.
  • 5 minutes toilet break if needed (only one).
  • 5 minutes medical time-out if needed (only one).

Court Switching

  • Players or teams switch ends of the court on odd games. This means they switch sides after the first game, and after every two games thereafter.

Code Violations

  • Code violations will be followed as per standard ATP and WTA norms.

Other Rules

  • All other standard rules set by ATP and WTA apply.
